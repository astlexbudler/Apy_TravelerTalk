post
post 앱에서 사용하는 HTML 템플릿이 정의된 디렉토리입니다.

작성방법:
1. base 디렉토리에서 HTML의 기본 구조를 상속받아 작성합니다. {% extends 'base/파일명.html' %}로 상속받아 사용합니다.
2. 기타 필요한 요소가 있다면 parts에서 작성한 HTML을 include하여 사용합니다. {% include 'parts/파일명.html' %}
3. 재사용 가능한 요소가 아닐경우, 주석과 함께 해당 파일 내에 작성합니다.

기타 궁금한 내용이 있다면 라이브 커뮤니티에 질문해주세요. 감사합니다.

필요 페이지:

/post
게시판 페이지. 이벤트, 공지사항, 리뷰, 여행지, 출석, 인삿말 게시판을 제외한 모든 게시판의 메인 페이지입니다.

/post/post_view
게시글 페이지. 이벤트, 리뷰, 여행지 게시글을 제외한 모든 게시글의 상세 페이지입니다.

/post/write_post
게시글 작성 페이지. 이벤트, 리뷰 게시글을 제외한 모든 게시글의 작성 페이지입니다.

/post/rewrite_post
게시글 수정 페이지. 이벤트, 리뷰 게시글을 제외한 모든 게시글의 수정 페이지입니다.

/post/attendance
출석 게시판 페이지입니다. 출석 메세지 작성과 출석 캘린더가 표시됩니다.

/post/greeting
인삿말 게시판 페이지입니다. 인삿말 메세지 작성이 가능합니다.

/post/notice
공지사항 게시판 페이지입니다. 게시글 접근 권한이 있는 관리자는 공지사항을 작성할 수 있습니다.

/post/event
이벤트 게시판 페이지입니다. 게시글 접근 권한이 있는 관리자는 이벤트를 작성할 수 있습니다.

/post/write_event
이벤트 작성 페이지입니다. 관리자만 접근 가능합니다.

/post/rewrite_event
이벤트 수정 페이지입니다. 관리자만 접근 가능합니다.

/post/event_view
이벤트 상세 페이지입니다. 이벤트 세부 정보를 확인할 수 있습니다.

/post/review
리뷰 게시판 페이지입니다. 사용자(user, dame)가 작성한 리뷰 게시글을 확인할 수 있습니다.
일간, 주간, 월간 탑 후기를 확인할 수 있습니다. 각 항목별로 10개의 게시글이 표시됩니다.
그 외 최신순으로 리뷰 게시글을 확인할 수 있습니다.

/post/review_view
리뷰 상세 페이지입니다. 리뷰 세부 정보를 확인할 수 있습니다.

/post/write_review
리뷰 작성 페이지입니다. 사용자(user, dame)만 접근 가능합니다.

/post/rewrite_review
리뷰 수정 페이지입니다. 사용자(user, dame)만 접근 가능합니다.

/post/travel
여행지 게시판 페이지입니다. 파트너가 작성한 여행지 게시글을 확인할 수 있습니다.
여행지 작성은 파트너 관리자 페이지에서 작성 가능합니다.

/post/travel_view
여행지 상세 페이지입니다. 여행지 세부 정보를 확인할 수 있습니다.

필요한 요소 및 contexts 정보는 해당 앱 및 사이트 맵 문서를 참고해주세요.
