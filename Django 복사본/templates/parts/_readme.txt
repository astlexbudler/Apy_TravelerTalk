parts
다른 HTML 템플릿들에서 재사용되는 HTML 요소가 정의된 디렉토리입니다.
작성된 HTML 파일은 include를 통해 다른 HTML 템플릿에서 사용됩니다.

작성방법:
1. 재사용되는 HTML 요소를 작성합니다. 예를 들어, 네비게이션 바, 푸터 등이 있습니다.
2. 다른 HTML 템플릿에서 include를 통해 사용합니다. {% include 'parts/파일명.html' %}로 사용합니다.
3. 지속적인 유지보수와 코드 가독성을 위해 주석을 작성해주세요.

기타 궁금한 내용이 있다면 라이브 커뮤니티에 질문해주세요. 감사합니다.
